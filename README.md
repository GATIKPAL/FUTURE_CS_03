# API Security Risk Analysis

## Overview
This project demonstrates a read-only API security risk assessment
performed on a public demo API.

## Tools Used
- Postman
- Browser DevTools

## Scope
- Public demo API
- Read-only requests
- No exploitation

## Methodology
- Documentation review
- Endpoint testing
- Risk identification
- OWASP API mapping

## Key Learnings
- Importance of API authentication
- Risks of excessive data exposure
- Real-world SaaS API security concerns
